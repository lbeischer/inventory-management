Design Decisions
+No database — zero setup, instant startup, no migrations
+Filter state in composable singleton — no prop drilling across views
+All API calls centralized in api.js — one place to update base URL or add auth headers
+Pydantic response_model on every endpoint — shape enforced automatically
-Writes (tasks, purchase orders) do not persist across server restarts
-No authentication — CORS open to all origins, demo only
